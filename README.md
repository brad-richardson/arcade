# Arcade

A collection of mini-games built with Godot 4.5 and GDScript. Designed for portrait-mode mobile play.

Games are self-contained folders under `games/` and auto-discovered by the hub at launch. Each game has a `meta.json` describing its name, description, coin rate, and other metadata.

Currently includes **Dodge** — avoid falling blocks, earn coins based on survival time.
